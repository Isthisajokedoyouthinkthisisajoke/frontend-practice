# Frontend Practice Project

A collection of frontend exercises featuring infinite scroll and animated text sliders with data fetching implementations.

## Features

### Pages 1-2: Infinite Scroller
- Smooth infinite scrolling content
- Auto-returns to top when reaching the end


### Pages 3-4: Animated Text Slider
- CSS keyframe-based text animations
- Two different data fetching implementations:

#### Page 3: Async/Await
- Fetches list items from a custom fake API
-  with `async/await`


#### Page 4: XMLHttpRequest
- Same functionality using XHR
